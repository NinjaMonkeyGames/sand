## WHAT IS THIS REPOSITORY FOR ?

This repository serves as a template for GameMaker Studio 2 Projects.

1. CI workflow YAML scripts to enforce repository standards.
2. Common document templates.
3. Branching and merge strategy.
4. Custom project integrity scripts to enforce project standards.

The purpose is to create a boiler-plate standard to create consistency across projects.

---

## WHO IS THIS REPOSITORY FOR ?

This repository is for any developer who wishes to integrate this with their own GameMaker project.
